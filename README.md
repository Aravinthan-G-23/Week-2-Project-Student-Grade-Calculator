Project Overview

The Student Grade Calculator is a beginner-friendly Python project that calculates a student's grade based on marks entered by the user. It demonstrates the use of functions, conditional statements, loops, and exception handling.

Features
Input student name and marks
Validate marks (0–100)
Handle invalid inputs using while loop
Use if-elif-else for grading
Display encouraging messages
Function-based design
Setup Instructions
Install Python 3.
Create project folder.
mkdir Student_Grade_Calculator
cd Student_Grade_Calculator
Run the program:
python grade_calculator.py
